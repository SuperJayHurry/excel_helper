# 前端文档

## 技术栈与结构
- 采用 JSP + JSTL 渲染，所有页面位于 `src/main/webapp/WEB-INF/jsp`
- 统一引用 `src/main/resources/static/css/main.css`，保持登录页与控制台页面的视觉一致性
- `fragments/header.jsp` 封装了页面头部与导航，管理员与教师视图共享

## 页面说明
### 登录页 `/login`
- 自定义卡片式登录，展示错误/退出提示，并标注默认账号信息
- 登录成功后根据角色自动跳转管理员或教师面板

### 管理员面板 `/admin/dashboard`
- “发布新的汇总任务”表单：包含任务名称、描述、截止日期、发送范围（全院/按系/自定义教师）、Excel 模板上传
- “已发布任务”列表：展示历史任务、模板下载与进度查看入口

### 任务详情 `/admin/templates/{id}`
- 展示任务元数据、模板下载、提醒与汇总按钮
- “教师反馈进度”表格：显示教师姓名、系别、邮箱、当前状态、提醒时间
- “附件与汇总”表格：列出各教师的提交时间、数据行数、附件下载；下方显示最新汇总信息及下载链接

### 教师面板 `/teacher/dashboard`
- 列出该教师收到的所有任务、截止日期、模板下载链接与当前状态
- 直接在表格内上传 Excel，提交后状态即时刷新

## 交互与提示
- 所有按钮和表单均提供基础校验与错误提示（如未上传模板）
- 通过 URL 参数 `?success`、`?submitted`、`?aggregated` 等方式向用户反馈操作结果
